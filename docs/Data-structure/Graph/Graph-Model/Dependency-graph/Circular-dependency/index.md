# Circular dependency

一旦存在 "circular dependency"，那么就可能存在:

1、deadloop

2、deadlock

因此我们遵循  [Acyclic dependencies principle](https://en.wikipedia.org/wiki/Acyclic_dependencies_principle) ，需要"break circle"。
