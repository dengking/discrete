# 关于本章

本章讨论了图的定义，构造，基本操作，以及图在计算机科学各个领域各种各样的应用。

本章内容主要来自《[Discrete Mathematics and Its Applications](https://www.amazon.com/Discrete-Mathematics-Applications-Kenneth-Rosen/dp/125967651X)》和维基百科、boost graph library。
