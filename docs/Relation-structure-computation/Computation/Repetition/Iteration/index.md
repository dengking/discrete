# Loop and iteration

iteration的含义是"迭代"，loop的含义是"循环"，大多数情况下两者含义是相同的，由于loop更加简单，所以本文中大多数地方都使用loop的说法。

2、[Iterative deepening depth-first search](https://en.wikipedia.org/wiki/Iterative_deepening_depth-first_search)
