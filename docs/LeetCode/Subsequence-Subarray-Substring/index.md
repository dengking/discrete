## 子序列(subsequence)

非连续，但是相对位置是保持不变的。

### 常用解法

常用解法:

1、如果"子序列、单调、最优化问题"，则可以使用滑动窗口

2、如果子序列，不满足单调性，但是具备最优子结构，在求最值问题的时候，可以使用动态规划算法

3、二分搜索
