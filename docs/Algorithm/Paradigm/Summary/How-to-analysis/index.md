# How to analysis

## systematic method: 罗列所有的可能性

1、问题的解空间、可能的组合、罗列所有的可能性、问题的解空间、组合分析

这是系统性地解决问题的第一步

我目前比较熟悉的分析解空间的方式是组合分析，这种分析方法并不能够代表所有，像动态规划的sequence类问题，它的解空间就不是组合分析的。

2、很多algorithm的优势在于它比较好地罗列了问题的所有可能性，典型的例子就是滑动窗口，先比与暴力搜索，它高效的多。
